# BDShare

social share

## Requirements

* Requires iOS 9.0 or later

## Installation

BDShare is available through [CocoaPods](https://cocoapods.org/pods/BDShare). To install it, simply add the following line to your Podfile:

```
use_frameworks!

pod 'BDShare'

```

## Usage

```
#import <BDShare/BDShareHeader.h>

```
